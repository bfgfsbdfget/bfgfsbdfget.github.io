## ‚öôÔ∏è Running markdown-site
After installing, follow these steps to run markdown-site:

1. Open your terminal or command prompt.
2. Navigate to the folder where you want to create your site. You can do this by using the `cd` command followed by the folder path.
3. Type the command:
   ```
   markdown-site init
   ```
   This command sets up your new markdown-site.

4. To start the server, type:
   ```
   markdown-site start
   ```
   This will make your site live locally for testing.

5. Open your web browser and go to `http://localhost:3000` to view your site.
